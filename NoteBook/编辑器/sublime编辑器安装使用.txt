2017.6.10     sublime text编辑器笔记


1.快速打开网页——添加到右键快捷方式
   win+R —— regedit(打开系统注册表) —— 搜索 shell ——  新建   sublime   新建  ——  command  ———   打开默认   修改路径(+空格-p空格--remote-tab-silent) "%1N"  括号里面可以不用
2.Linx操作系统的Vi编辑器
    1.开启Vi功能   安装插件vintage     
    2.Vi编辑器模式
       1.输入模式(光标是竖着的)
          1.写内容
       2.命令模式(光标是横着的)
          1.操作快捷键
       3.末行模式
          1.：w      ——保存不退出
          2.：wq|x     ——wq|x是保存后退出
          3.：q!       强制不保存退出
       4.ctrl+shift+t(windows 快捷键)  快速已关闭的标签
       5.字体大小   
        增大：  ctrl+“=”
        减小：  ctrl+shift+“=”
    3.模式切换
       1).修改默认打开的编辑模式
          修改Packages\Vintage中的 Preferences.sublime-settings 中的"vintage_start_in_command_mode": true
       2).命令——>末行模式

       3).命令——>输入模式
		  iosa(IOSA)
		  i:在选取的目标的前面插入
		  o:在选取的目标的下一行输入
		  s:在选取的目标的位置输入（删除当前字符）
		  a:在选取的目标的后面输入
		  I：行首输入
		  O：上一行输入
		  S：删除整行
		  A：行尾输入
       4).输入——>命令
          ESC
       5).命令——>末行
          shift+":"
       6).末行——>命令
          ESC(2次)
    4.Vi编辑器常用命令
       1.ctrl+LB可以多选行
       2.块移动
            右：选择后tab
            左：选择后shift+tab
       3.光标移动
	         H                 左
	         J                 下
	         K                 上
	         L                 右
       4.删除
	         dd               删除本行
	         r                替换
	         u  ctrl+z        撤销
	         ctrl+y           恢复
	         dg               从当前行删除到最后一行
	         d+number+g       从当前行删除到number行
	         dw               删除单词(英文)，中文情况删除后面的一串字符
	         d^               从当前字符删除到行首
	         d$               从当前字符删除到行尾
	         number+dd         从本行开始删除3行
       5.跳转
	         shift+"G"         到最后一行
	         G                 到第一行
	         number+"G"       到第number行
       6.复制
	          yy               复制
	          number+yy        复制number行
       7.粘贴
	          p(小写)                粘贴到下一行
	          P(大写)                在当前行的前一行粘贴
	          p(小写)         粘number次到下一行
	          P(大写)         粘number次到上一行
       8./+内容           查找“内容”，查找多个按N键
       9.v                从当前视图选择内容
       10.x               删除当前字符

3.快速注释(智能注释)
  ctrl+/cBlockr
4.emmet插件
   1.!+tab 
         快速生成H5基础代码
   2.div>div>div>div>div>div>div>div>
         快速生成嵌套的div
   3.div#ID.class{内容}*number
         快速生成含有内容选择器的元素,当内容为$$时，默认从0开始排序，*number：重复前面的带啊number次
5.常用的HTML插件
    ctrl + shift + p             命令面板
    Emmet         快速生成H5代码
    JsFormat      JS代码格式化
       使用：鼠标右键JsFormat或者ctrl+Alt+F
    SideBar Enhancements     增添左侧栏中的快捷功能
    SublimeCodeIntel        代码提示、自动补全
    DocBlockr               智能注释工具
    Better Completion       JavaScript jQuery  bootstrap 代码的自动补全
    AUTOPREFIXER/Sublime Prefixr插件    CSS3私有前缀自动补全插件(需要node.js环境)
    ColorPicker      
    AutoFileName        自动补全文件路径
    vintage             Vim编辑器
    ChineseLocalizations   中文
  
 6.sublime常用快捷键
    ctrl + shift +上   与上面一行互换 
    ctrl + shift +下   与下面一行互换
    ctrl + Enter       光标之后插入行
    Ctrl + shift       光标之后插入行
    Ctrl + shift + K  删除整行
    Ctrl + shift + D  复制本行到下一行
    Ctrl + M          光标移动到括号的开始或者结束
    Ctrl + shift + M  选择光标所在的括号内的所有内容
