1.tcp/ip协议（族）
    tcp/ip不是值得一个两协议，而是一大堆协议族，用于规范网络中数据传输

    链路层
        网络接口
    网络层
        ICMP、ARP、RARP、IP、IGMP
    传输层
        TCP、UDP
    应用层
        应用进程

2.端口
    ip地址定位主机、端口定位主机中的进程
    netstat -an  查看端口信息

    知名端口（1024-65535）：     
                80  HTTP服务器  
                21  分配给FTP服务器

3.ip地址
    用于定位主机
    ip地址=网络号+主机号
    主机号不能为0     主机号为0的时候 表示的是这个局域网的网络号
    主机号不能为255   主机号为255的时候表示广播地址


4.socket
    套接字---通过网络的进程之间的通信方式

    python创建套接字

        //创建TCP通信（慢、但是不会丢失数据）
        import socket
        s=socket.socket(socket.AF_INET,socket.SOCK_STREAM)

        //创建UDP通信（快、但是可能会丢失数据）
        import socket
        s=socket.socket(socket.AF_INET,socket.SOCK_DGRAM)



    python套接字绑定端口
        如果应用程序没有绑定端口，那么每次运行的时候都会由系统随机分配一个端口
        绑定ip地址与端口
            socket.bind("ip",端口)
            
            发送方如果只是发送，则不必要绑定IP地址
            接受方必须绑定IP地址，如果id地址为空 则绑定本机所有IP地址


    python套接字说明    
        python3中的网络套接字发送数据之前加一个b
        使用TCP发送数据，在发送数据之前建立连接，之后发送的数据不需要再次建立连接
        使用UDP发送数据，在每次发送数据的时候都需要说明要发送的目的地址
