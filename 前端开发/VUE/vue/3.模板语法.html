• 插值

    • 示例: <span>Message: {{ message }} </span>

    使用 v-once 指令,使得内容一次性插值,之后不在响应式更改:{{ message }}

    这里不会解析成HTML元素: {{ msg }}

  • 属性

    • 双括号的方式不适用于HTML标签上的属性,对于属性应当使用v-bind指令

    <span v-bind:id="wrap1">元素内容</span>

    对于布尔属性: disabled,autoplay,preload等,只要存在就为真,
    但是给定的值为假的时候: null,undefined,false等, 这些属性不生效




  • 双括号中的表达式

    • 双括号中的表达式应该是一条JavaScript的表达式,不能是语句,也不能是控制结构循环结构

    双括号中只能访问部分全局变量,例如: Math,Date 尽量不要在括号中使用全局变量

    加法: 2+2={{2+2}}

    减法: 4-2={{4-2}}

    乘法: 2*2={{2*2}}

    除法: 4/2={{4/2}}

    三目: {{2===2?"2===2":"2!==2"}}

    全局对象:{{new Date().toLocaleString()}}

  • 指令:指令是指带有v-前缀的特性,指令期待单个表达式

    • 指令的作用是,当其表达式的值改变的时候,将其产生的影响响应式的作用域dom

    Vue中的指令有(14):

    • v-text
    • v-html
    • v-show
    • v-if
    • v-else
    • v-else-if
    • v-for
    • v-on
    • v-bind
    • v-model
    • v-slot
    • v-pre
    • v-cloak
    • v-once

    参数

    Vue中的 v-on 以及 v-bind 都需要一个参数,例如:

    <a v-bind:href="url">...</a>

    这里的href就是指令的参数,将这个参数于url绑定

    动态参数

    <a v-bind:[attributeName]="url"> ... </a>

    以上的attributeName是一个Vue对象中的属性,这里绑定的属性,根据attributeName的具体值而改变
