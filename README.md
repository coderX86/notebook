# NoteBook

## 说明

`@hec9527`的笔记文档，依赖于`docsify`构建， [docsify 文档](https://docsifyjs.netlify.app/#/zh-cn/)

## 安装

```shell
npm i -g docsify
```

## 使用

```shell
# 克隆仓库
git clone https://github.com/hec9527/notebook && cd notebook

# 安装依赖
npm i docsify

# 启动服务
npm run docs
```

> 文档`docs`目录的`.nojekyll`文件，是为了防止`github`忽略以下划线开头的文件
