# VsCode

VsCode 是微软开发的一款基于`Electron`开源的文本编辑器，它不同于 IDE，它在保证轻量化的同时，极大的提升了编码速度以及使用体验，内置的插件商店可以将这款文本编辑器武装成一款`真·IDE`  
优点：

1. 开源免费
2. 可定制化程度高
3. 内置 Git
4. 支持 Debug
5. 可扩展
6. 智能
7. 轻量化
8. ...

**_重新定义编辑器_**

## vs code 常用快捷键

| 快捷键                   | 叙述                   |
| ------------------------ | ---------------------- |
| F1                       | 命令面板               |
| F2                       | 重命名                 |
| Ctrl + Tab               | 文件切换               |
| Home / End               | 转到行首/行尾          |
| Ctrl + Home / Ctrl + End | 转到文件头/文件尾      |
| F12 or Ctrl + click      | 转定义                 |
| Ctrl + Alt + click       | 转定义（切出新编辑器） |
| Alt + F12                | 查看定义               |
| Shift + F12              | 查看引用               |
| Alt + Up / Alt + Down    | 上下移动一行           |
| Shift ＋ Alt + F         | 代码格式化             |
