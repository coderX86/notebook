## Web新技术

- WebAssembly
  - 用于辅助JavaScript、将其他语言编译成可以在浏览器运行的二进制文件，目前支持C++、Rust

- Web Components
  - Web Components是类似于React、Vue的开发方式，JS原生支持，不在需要而外的库文件，核心概念是：Custom Elements，Shadow DOM，HTML templates
