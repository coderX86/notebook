# 说明  <!-- {docsify-ignore-all} -->

`@hec9527`的笔记文档，依赖于`docsify`构建， [docsify文档](https://docsifyjs.netlify.app/#/zh-cn/)


## 安装

```shell
npm i -g docsify
```

## 使用

```shell
# 克隆仓库
git clone https://github.com/hec9527/notebook
cd notebook

# 初始化   只需要在首次安装时运行该命令
docsify init docs

# 启动服务
docsify serve docs
```
