Numpy库
    numpy 中的数组都是同一种数据类型的，方便确定数组的大小
    array模块：类似于C语言的一维数组
    ndarray：numpy提供的用于存储单一数据类型的多维数组

    数据类型
        bool        1位存储布尔值
        inti        由平台决定精确度 32/64
        
        int8        整数 -128 ~ 127
        int16       整数 -2^16 ~ 2^16-1
        int32       整数 -2^32 ~ 2^32-1
        int64       整数 -2^64 ~ 2^64-1

        uint8       无符号整数 0 ~ 255
        uint16      无符号整数 0 ~ 65535
        uint32      无符号整数 0 ~ 2^32-1
        uint64      无符号整数 0 ~ 2^64-1

        float16     半精度浮点数    1位正负  5位指数  10位尾数
        float32     单精度浮点数    1位正负  8位指数  23位尾数
        float64/float 双精度浮点数  1位正负  11位指数 52位尾数

        complex64   复数    分别使用两个32位浮点数表示实部、虚部
        complex128/complex  复数   分别使用两个64位浮点数表示实部、虚部

    两种对象
        ndarray      N-dimensional Array Object 多维数组
        ufunc        Universal Function Object
    数组属性 ( ndarray ) 
        ndim    
            返回int 数组的维数
        shape
            返回tuple ， 数组的行列的长度       可以设置切换形状 但是不能修改元素的总个数
        size
            int 表示数组的元素的总数
        dtype
            data-type   数组中的数据类型   默认为浮点型
        itemsize
            int 表示数组中的每一个元素的字节大小  1字节=8bit
    创建数组
        使用python序列创建
            numpy.array(obj,dtype=None,copy=True,order='K',subok=False,ndmin=0)
                obj     想要创建的数组  必须
                dtype   数组的数据类型   缺省自动判断
                ndmin   int 数组的最小的维数  默认为None
        使用numpy中的函数创建   --高效
            numpy.arange(m,n,k) 
                同python中的range用法
            numpy.linespace(m,n,k)
                从m到n的等差一维数组
            numpy.logsapce(m,n,k)
                从m到n的等比一维数组
            numpy.zero((m,n))
                创建一个m行n列的二维数组，也可以是多个参数返回一个多维数组
            numpy.eye(k)
                创建一个k行k列的对角线上的元素为1的二维数组
            numpy.diag([1,2,3,4])
                创建一个对角依次为1,2,3,4的二维数组
            numpy.one((m,n))
                创建一个m行n列的二维全部元素都是1的数组，多个参数返回一个多维的数组
    随机数
        numpy.random.random(100)
            生成0到1不包含1的随机数100次 返回这个数组
        numpy.random.rand((m,n))
            生成m行，n列的二维均匀分布的随机数，值为0-1不包含1   多个参数可以生成多维随机数    
        numpy.random.randn((m,n))
            生成m行，n列的二维正态分布的随机数，           多个参数可以生成多维随机数    
        numpy.random.randint(low,hight=None,size=None,dtype="1")
            生成最小为low 最大为height 大小为size的指定 行列的多维随机数 ，size指定大小 int型   可以是整数也可以是包含多个变量的int型的tuple 
            可以不用给定上限，但是最好给一个

    数组索引
        一维数组索引
            同python中的序列的索引，支持单个索引、切片、倒序……    步长为负数的时候开始下表需要大于结束下表
            可以使用下表修改值 
        多维数组索引
            多维数组中每一个维度都有一个索引，各个维度之间的所引值使用 逗号 隔开
                arr[1,2]    #索引arr的第二行第3列        
            多维数组也可以索引切片
                arr[1,2:4]  # 索引arr的第2行第3-4列   #不包含切片的结尾
                arr[0:4,3]  # 索引arr的第1-4行的第3个
                arr[0:2,0:2]  # 索引arr的第1-2行的第1-2个
                可以通过上面的切片集体赋值  赋值一个那么全部等于一个值   多个如果形状相同则赋对应的值 如果行可以对应列也就可以整列赋值  如果数量不匹配则报错

    数组的变形
        numpy.reshape(m,n)
            将数组转换为m行，n列的数组     m*n=原来数组中元素的个数
        numpy.flatten('F')
            将数组纵向展平    没有'F'横向展平   
        numpy.ravel()
            将数组横向展平
        numpy.hstack((arr1,arr2))
            将两个数组横向组合，将arr2每一行添加在arr1之后，  arr2 与arr1 必须由相同的行数
        numpy.vstack((arr1,arr2))
            将两个数组纵向组合，将arr2每一列添加在在arr1之后， arr2 与arr1必须有相同的列数
        numpy.hsplit(arr,x)
            将arr横向分隔为x个数组   每个数组为原看来的 len[arr[0]/x]列   如果有小数报错
        numpy.vsplit(arr,x)
            将arr纵向分隔为x个数组    x要求 同上
        numpy.split(arr,x,axis=None)
            在axis=1时横向切割，asix=0时纵向切割
    矩阵计算
        numpy中矩阵时ndarray的子类  矩阵对象继承与二维数组对象

        创建矩阵对象
            如果使用 matrix 或者使用ndarray对象创建矩阵，则不会生成副本 等效与 matrix(data,copy=False)
            matr=numpy.mat("1 2 3;4 5 6;7 8 9")
                创建一个3行3列的矩阵
                创建矩阵元素之间用空白隔开  不同行使用 分号隔开
            matr=numpy.matrix([[1,2,3],[2,3,4],[3,4,5]])
                创建一个3*3的矩阵
                使用二维数组来生成一个矩阵
            mat函数使用数字生成矩阵
            bmat函数使用矩阵生成矩阵    -  参数为矩阵  而不是整数  cleartr
